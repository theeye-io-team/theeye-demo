TheEye test server

/scraper

this endpoint always return the stored query string value as JSON. startup value is "success"

  • GET : get the stored value
  • PUT : update the stored value using query string

/process

this endpoint creates a named process in the system that stores a unix timestamp every second in the file /tmp/theeye-${name}-process.log

  • GET : return the status of the process name ?name=name
  • POST : create a process with the provided name ?name=name via query string
  • DELETE : stop a process with the provided name ?name=name via query string
